Tags HTML5

As tags são marcas semânticas usadas para organizar e dar sentido as coisas existentes em sites web. Confira algumas possibilidades que elas proporcionam aos desenvolvedores web.

  • Parágrafos
  • Quebra de linha
  • Adicionar imagem
  • Títulos e seus níveis
  • Evidenciar ênfase
  • Evidenciar força semântica
  • Fazer citação
  • Escrever código em sua página web
  • Deixar o texto pré formatado
  • Adicionar lista
  • Adicionar hiperlink
